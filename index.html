Terminalicious by mbcharbonneau

A handy OS X menu icon application for launching commands in Terminal.app and iTerm.

Terminalicious is a handy menu icon application for launching commands in Terminal.app (or iTerm, if you prefer). Drag and layer the transparent input window anywhere on the screen, or simply hide it when you don't need it. Terminalicious saves a history of your commands (either alongside the input window, or in the menu), so you don't have to constantly re-type frequently used commands. Terminalicious is made for people who frequently open a terminal session to run a single command, such as starting a backup or monitoring a log file. Just think of it as a convenient way to start a Terminal session.
